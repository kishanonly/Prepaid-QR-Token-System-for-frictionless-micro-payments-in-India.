# Digital Change – Prepaid QR Token System

## Problem
Small payments still have friction in UPI.

## Solution
Prepaid one-time-use QR payment tokens.

## Use Cases
- Parking
- Tea stalls
- Railway tips
- Hotels
- Delivery
- Children payments

## Benefits
- Faster payments
- Digital change replacement
- Offline-friendly
- Child-friendly

## Security
- One-time token
- Expiry
- Fraud protection
